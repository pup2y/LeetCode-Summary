# 1.滑动窗口与双指针

## 01.定长滑动窗口
题目中标明固定长度k  
```cpp
int ans = 0, statistic = 0; //初始化答案，统计量
for(int i = 0; i < 容器.size();i++){ //进入循环窗口
  update(statistic,容器[i]); //引入索引为i的元素 更新统计量状态
  if(i<k-1){
    continue;     //若i<k-1则重复第一步
  }
  update(ans);   //更新答案
  update(statistic,容器[i]); //除去索引为i-k+1的元素 更新统计量状态
}
return ans;
```
